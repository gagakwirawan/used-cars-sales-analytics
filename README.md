## About the project
So, the premise of this project is to analyze (by volume) the sales of used cars from a certain company in Europe.
I wanted to know how the sales went so far and whether there are certain features that affect the sales.
I used python libraries like numpy, pandas, seaborn, and matplotlib in general for this project.
I also used some statistical overview rather than just putting in charts to pull a conclusion.
Even though I wasted most of my time during preprocessing.
Oh, I forgot to mentioned that the dataset that I used, I got it from https://doi.org/10.6084/m9.figshare.19586296
(I even put a txt file consisting just the link so that if people did not see my readme, they can see it directly on their files).
